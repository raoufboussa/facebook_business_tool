How We Can Use This tool

Our service represents a dedicated tool which supports a marketing manager by gathering all the necessary information in a single place thus he can be informed of the business's state of a client (pages and advertising campaigns) ,also it allows to analyze them and detect information allowing to build for each customer a digital strategy according to the different circumstances and then improve it in an effective way.
This tool is used after going through some steps.

Step 1: Link The Social Account

First you have to link the facebook account to this application, so we fill out this form with the information of the facebook account:

  • First you have to link the user who is already registered.
  • Choose the facebook provider.
  • Copy the Facebook Uid of the account.

Step 2: Link The Social Application

After we linked the facebook account, now we have to link the business application linked to this account, it is the application with which we manage our business, in this application we find tools like graph explorer, it is is the tool which gives access to the data (publications, comments, interactions ... etc.) of the pages and ad accounts linked to this facebook account.

  • Choose the facebook provider.
  • Give the application a name, for example "Facebook App".
  • Copy the client Id from the Facebook application.
  • Copy the secret key of the Facebook application.

Step 3: Link The Social Token Of Application

Social Token is the most important code, it allows us to make http requests to get the data that we want, so to finalize the configuration of the social application you have to extract the token

  • Link the application that we have already created.
  • Linked to the facebook account that we have already created.
  • Finally copy the application token.

Step 4: remplire la base de donnée avec les metrics nécessaire pour le bon fonctionnement de l'outil

The Pages API allows apps to access and update a Facebook Page's settings and content, create and get Posts, get Comments on Page owned content, get Page insights, update actions that Users are able to perform on a Page, and much more.
You have to run this script to fill the database with the necessary metrics.

  • Run : "python manage.py makemigrations" in the terminal.
  • Run : "python manage.py migrate".
  • Run : "python manage.py shell".
  • Copy the script below and past in the shell run to fill the database.
  • run to fill the database.
